# PRD - Windows 桌面便签应用

## 项目概述

开发一个 Windows 桌面便签应用，使用 React + Tailwind 技术栈，通过 Electron 打包为桌面应用。

## 技术栈

- **前端框架**: React
- **样式**: Tailwind CSS
- **桌面包装**: Electron
- **数据存储**: 本地文件存储 (JSON 文件)

## 功能需求

### 1. 窗口特性
- 窗口悬浮在桌面最顶层 (always on top)
- 黑色半透明背景
- 支持拖动窗口边缘调整大小
- 无边框设计，仅显示便签内容

### 2. 便签管理
- 支持创建多个便签窗口
- 每个便签独立编辑
- 便签内容本地文件持久化存储

### 3. 快捷键功能
- 全局快捷键显示/隐藏便签 (默认: Ctrl + `)
- 支持快捷键录制自定义
- 每个便签可设置独立的显示/隐藏快捷键

### 4. 界面设计
- 极简风格，仅显示用户输入的文字内容
- 无冗余图标或按钮
- 黑色半透明背景 (透明度可配置)

## 非功能需求

- 开机自启动 (可选)
- 窗口位置和大小记忆 (可选)

## 存储结构

```json
// data/notes.json
{
  "notes": [
    {
      "id": "uuid",
      "content": "便签内容",
      "hotkey": "Ctrl+`",
      "x": 100,
      "y": 100,
      "width": 300,
      "height": 200,
      "opacity": 0.8
    }
  ]
}
```

## 后续可扩展功能

- 字体大小调整
