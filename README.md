# 桌面便签 (Desktop Note)

Windows 桌面便签应用，支持多个便签、快捷键显示/隐藏、窗口置顶。

## 功能特性

- **多便签管理**: 创建、切换、删除多个便签
- **窗口置顶**: 可切换窗口置顶状态
- **透明背景**: 支持透明度调节
- **字体大小**: 支持调节字体大小 (12-96px)
- **自由调整**: 支持自由调整窗口大小和位置
- **数据持久化**: 便签内容自动保存到本地
- **快捷键控制**: 全局快捷键显示/隐藏便签 (默认 `Ctrl+\`)

## 数据存储位置

便签数据存储在以下文件：

- `notes.json` - 便签内容（ID、内容、位置等）
- `window-config.json` - 窗口配置（尺寸、位置、快捷键、透明度、字体大小）

存储路径: `%LOCALAPPDATA%\DesktopNote` (Windows)

## 安装与运行

```bash
# 安装依赖
npm install

# 构建生产版本
npm run build

# 启动应用
npm start
```

## 使用说明

### 基础操作

- **新建便签**: 点击底部「+ 新建」按钮
- **切换便签**: 点击顶部便签标签
- **删除便签**: 点击底部「×」关闭按钮 (最后一个便签会清空内容)

### 窗口控制

- 顶部按钮: 最小化、最大化、关闭
- 底部「📌」按钮: 切换窗口置顶

### 设置

点击底部「⚙」打开设置面板：

- **透明度**: 调节便签背景透明度
- **字体大小**: 调节便签字体大小 (12-96px)
- **快捷键**: 点击按钮后按下新快捷键
